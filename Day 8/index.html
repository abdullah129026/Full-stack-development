CSS Syntax

CSS stands for Cascading Style Sheets. It is used to style and layout web pages.

Selectors

Selectors are used to target HTML elements to apply styles to them.

Specificity

Specificity determines which CSS rule is applied when there are multiple rules that could apply to the same element.

The Box Model

The box model describes the rectangular boxes that are generated for elements in the document tree and how they are laid out.
